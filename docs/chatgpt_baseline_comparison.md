# ChatGPT baseline comparison

The baseline module simulates what a generic ChatGPT prompt would likely find: vague CTA, missing proof, generic language, and broad founder/operator resonance.

The Locad validator adds:

- ICP coverage report
- ICP ID-backed evidence
- pain/value/do-don't traceability
- proof-gap diagnostics
- buying committee objections
- benchmark and gate output
